How to Link Existing IEC with DGFT Portal

If you already have an Import Export Code (IEC) issued earlier, you must link it with your DGFT portal login to manage IEC details online.

Linking IEC allows exporters and importers to:

  • Access IEC dashboard
  • Modify IEC details
  • Apply for DGFT schemes
  • Track application status
  • Manage users associated with IEC

Requirements to Link IEC

Before linking your IEC ensure the following:
  • Valid Digital Signature Certificate (DSC)
  • Registered DGFT login account
  • Valid mobile number and email ID
  • IEC number issued by DGFT

Step-by-Step Process to Link IEC

Step 1 — Visit DGFT Portal

Open the DGFT website:

https://dgft.gov.in

Step 2 — Register on DGFT Portal

If you do not have an account, register first:

Login → Register → Register as Importer / Exporter

Provide mobile number and email ID and verify OTP.

Step 3 — Login to Dashboard

Login using your registered email ID and password.

Step 4 — Navigate to Link IEC

Go to the following section:

Services → Link IEC

Step 5 — Enter IEC Details

Enter your IEC number and click Validate.

Step 6 — Authenticate IEC

Authentication can be done using:

  • Digital Signature Certificate (DSC)
  • Aadhaar based e-Sign

Step 7 — IEC Successfully Linked

After successful validation, the IEC will be linked to your DGFT account and will appear in your dashboard.

Primary User vs Secondary User

Primary User

Primary user has full access to IEC profile.

Capabilities include:
  • Submit applications
  • Manage IEC profile
  • Add or remove users
  • Access DGFT schemes

Secondary User

Secondary users have limited access.

Capabilities include:
  • Draft applications
  • Save application data
  • Assist with compliance filings

Important Notes

  • Only one IEC can be linked with a single user profile.
  • DSC must be registered on DGFT portal before validation.
  • Once linked, IEC details can be managed online.
